MVP Details----

Homepage:
Includes listings within 50 miles (default) of the GPS location.
Each listing comes with a title, description, price, location info, and contact info with a call button.
The listings should be sorted by popularity (most views) if possible.

Search:
Search bar, Category drop down, Radius, Search Button.
Text that states which cities are included.
